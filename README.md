# Rails-Job Portal

A simple Job-Portal web application using Rails 6.

### Purpose

### Basic Setups

* Ruby version - 3.0.0
* Rails Version - 6.1.3.1
* GitHub

### Important Library

* gem 'devise' - User Authentication
* gem 'Figoro' - Set Environment Variables
* gem 'JWT'    - API Authentication
* Bootstrap    - Styling
